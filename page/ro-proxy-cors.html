RO Proxy Cors

RO Proxy CORS Activați CORS (partajarea resurselor încrucișate) pentru orice adresă URL cu această API simplă și converentă!

Demo: https://showchat.gq (așteptați un pic pănă se incarcă)

Proxy invers cu anteturile CORS

Utilizare: https://showchat.gq/ +url

Fork on GitHub

Folosind Proxy-ul pentru a merge mai departe CORS și utilizați HTTPS pentru apelurile API ShoutCast

Ei bine, dacă aveți un player web și doriți să transmiteți un flux sigur, ceea ce înseamnă că serverul dvs. de web are certificat SSL.

Apoi, faceți streaming cu https url în loc de http-ul obișnuit. Din păcate, Shoutcast nu va fi transmis în mod corect dacă se folosește cu HTTPS.

Aceasta înseamnă că există posibilitatea ca numai audio, să se transmită în mod corect în timp ce opera de artă a albumului sau chiar numele titlului să nu se afișeze deloc.

Aici este locul în care serverul CORS PROXY joacă un rol important.

Dezvoltatorii jucătorilor dvs. web vor crea acest server proxy și îl vor lega cu URL-ul dvs. de streaming ShoutCast pentru a transmite în siguranță ...
